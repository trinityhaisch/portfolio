### Update Your Info (5 minutes)

1. **Name & Title** - Edit `app/page.tsx`:
   - Line ~25: Change "Your Name"
   - Line ~30: Change "Software Engineer"
   - Line ~35: Update intro sentence

2. **About Page** - Edit `app/about/page.tsx`:
   - Line ~20-25: Replace bio text
   - Line ~50: Update LinkedIn URL
   - Line ~70: Update email address

3. **Footer** - Edit `components/Footer.tsx`:
   - Line ~20: Update name
   - Line ~30: Update email

4. **Projects** - Edit `data/projects.ts`:
   - Replace example projects with your own
   - See README.md for detailed instructions
